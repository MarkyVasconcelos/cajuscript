# Arithmetic Operators #

All arithmetic operators allowed.

### Addition ###
```
     x = 0 + 1
     x += 1
```

### Subtraction ###
```
     x = 0 - 1
     x -= 1
```

### Multiplication ###
```
     x = 0 * 1
     x *= 1
```

### Division ###
```
     x = 0 / 1
     x /= 1
```

### Modulus ###
```
     x = 0 % 1
     x %= 1
```

_**Not is supported "++" and "--"! Use "+=1" and "-=1".**_
